
**DELIVERY(CONTAINER(TRIGGER + PAYLOAD + DECOY))** :

- **Delivery** -> Technique utilisée pour livrer le package à la victime.
    
- **Container** -> Format de conteneur utilisé pour emballer les fichiers.
    
- **Trigger** -> Moyen de déclencher l'exécution du payload.
    
- **Payload** -> Code malveillant à éxecuter.
    
- **Decoy** -> Fichier montré à la victime après interaction avec le Trigger.
