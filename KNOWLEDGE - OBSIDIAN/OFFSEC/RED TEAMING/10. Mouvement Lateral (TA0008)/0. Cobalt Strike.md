

Cobalt strike à 2 commandes permettant le mouvement lateral

- jump
- remote-exec


## Jump

Permet d'avoir une session interactive.

Automatise ces actions :

- Upload un payload sur la target.
    
- Execute le playload à distance en mémoire.
    
- Se connecte au nouveau beacon de session.
    
- Supprime le payload de la target.


```
jump [exploit] [target] [listener]
```


## Remote Exec

Permet d'executer une commande à distance. (supporte WMI)

```
remote-exec [method] [target] [command]
```
