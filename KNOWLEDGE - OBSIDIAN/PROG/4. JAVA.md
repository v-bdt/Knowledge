

# INSTALL OPENJDK

installer openjdk 11

```sh
sudo apt install openjdk-11-jdk
```

définir openjdk 11 comme version à utiliser

```sh
export JAVA_HOME=/usr/lib/jvm/java-1.11.0-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```


# COMPILER / DECOMPILER


Décompiler le .jar

```
mkdir jar_content
cd jar_content
jar xf ../monitoringClientEXPLOIT.jar
```

Passer une .class au format .java afin de pouvoir la modifier

```
cp com/path/to/file.class com/path/to/file.java
```

Compiler un .java en .class

```
javac -cp ".:*" com/path/to/file.java
```

```
javac Main.java
```

update le .jar file avec la nouvelle classe

```
jar uf ../monitoringClientEXPLOIT.jar com/path/to/file.class
```


# Build projet avec MAVEN

MAVEN est un outil de build de projets automatisé créé par Apache.

1. installer maven

```sh
sudo apt install maven
```

2. build du projet à partir de la racine de celui-ci.

```sh
mvn clean package -DskipTests -X
```

si BUILD FAILURE -> [ERROR] Failed to execute goal org.apache.maven.plugins:maven-assembly-plugin:3.7.1:single (make-assembly) on project ysoserial: Error reading assemblies: No assembly descriptors found. -> [Help 1]
org.apache.maven.lifecycle.LifecycleExecutionException: Failed to execute goal org.apache.maven.plugins:maven-assembly-plugin:3.7.1:single (make-assembly) on project ysoserial: Error reading assemblies: No assembly descriptors found.

Mettre à jour le descriptor dans le pom.xml

```xml
<descriptors><descriptor>assembly.xml</descriptor></descriptors>
```

au lieu de 

```xml
<descriptor>assembly.xml</descriptor>
```
